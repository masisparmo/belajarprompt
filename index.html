Pengaturan API Key

Masukkan satu atau lebih API key, pisahkan dengan koma jika lebih dari satu. Key akan digunakan secara berurutan.

Cara Mendapatkan API Key Gemini (Gratis)

  1. Buka Google AI Studio dan login dengan akun Google.
  2. Pilih menu "Get API key" di panel sebelah kiri.
  3. Klik tombol "Create API key".
  4. Pilih project yang ada atau buat baru, lalu klik "Create API key in new project".
  5. Salin (Copy) kode unik yang muncul dan tempel (Paste) di atas.

Masukkan satu atau lebih API key, pisahkan dengan koma. Ini akan digunakan jika permintaan ke Gemini gagal.

Cara Mendapatkan API Key Groq (Gratis)

  1. Buka Groq Console dan login.
  2. Di panel menu sebelah kiri, klik opsi "API Keys".
  3. Klik tombol "Create API Key".
  4. Beri nama (misal: "PanduanAI"), lalu klik Submit/Create.
  5. Salin (Copy) kode yang berawalan gsk_ dan tempel (Paste) di atas.
